# Tutorials

You can learn how to use OpenGateLLM with those tutorial Jupyter notebooks:

- [Advanced RAG](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/advanced_rag_pipeline.ipynb): how to download and parse PDF documents, extract embeddings, create collections and add a document, make a document-based prompt to a model.
- [Audio Transcription](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/audio_transcriptions.ipynb): how to discover available speech recognition models, download audio files from the internet, and convert speech to text using automatic speech recognition capabilities.
- [Chat Completion](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/chat_completions.ipynb): streamed and non-streamed text generation using OpenAI-compatible endpoints.
- [Models](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/models.ipynb): how to discover and retrieve information about available AI models through OpenGateLLM `/v1/models` endpoint, displaying model types, context lengths, and other specifications.
- [Optical Character Recognition (OCR)](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/ocr.ipynb): how to perform OCR on PDF documents using image-text-to-text models to extract text from visual content.
- [Search and RAG](https://github.com/etalab-ia/OpenGateLLM/blob/main/docs/tutorials/search_and_rag.ipynb):  how to build a complete search and RAG system using OpenGateLLM, including creating collections, importing documents, and performing semantic search with vector databases.
